# Human Insulin BLAST Analysis

## Project Overview

This project focuses on the bioinformatics analysis of the Human Insulin protein sequence using NCBI BLAST (Basic Local Alignment Search Tool). The objective is to identify homologous protein sequences, evaluate sequence similarity, and study evolutionary conservation across different species.

## Objectives

- Retrieve the Human Insulin protein sequence from UniProt.
- Perform BLASTP analysis using NCBI BLAST.
- Identify homologous protein sequences.
- Analyze sequence identity, similarity, alignment score, and E-value.
- Interpret the biological significance of the obtained results.

## Tools and Databases Used

- UniProt Database
- NCBI BLASTP
- NCBI Protein Database

## Methodology

1. Obtained the Human Insulin protein sequence (UniProt ID: P01308).
2. Submitted the sequence to NCBI BLASTP.
3. Compared the query sequence against protein databases.
4. Examined alignment results, identity percentage, and E-values.
5. Documented findings with screenshots and analysis.

## Results

The BLAST analysis identified several homologous insulin proteins from different organisms, demonstrating a high degree of sequence conservation. The top matches showed significant sequence identity and extremely low E-values, indicating strong evolutionary relationships.
